# front-end-challenge
A aplicação foi escrita utilizando Node.js

### Caso seja a primeira vez que vai executar:
```
npm install
```

### Para iniciar a aplicação
```
npm start
```
```
yarn start
```


### O que foi utilizado

- create-react-app (react, webpack, babel, eslint, etc)
- node-sass (CSS)
- prettier (identação dos códigos)
